# rag_chatbot
Agentic RAG chatbot exhibition project

To install the dependencies, run the following in the project folder:

pip install -r requirements.txt

For generating Gemini API key to test the project:

1. go to https://aistudio.google.com/app/api-keys
2. click on "Create API key"
3. create a .env file to the project folder
4. add this line to the .env file: GOOGLE_API_KEY=<your_api_key>
